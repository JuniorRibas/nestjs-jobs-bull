## Description
 

Project started using  [Nest](https://github.com/nestjs/nest), HUB mkt gruplira
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start && docker-compose up --build

# watch mode
$ npm run start:dev && docker-compose up --build

# production mode
$ npm run start:prod && docker-compose up --build
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
